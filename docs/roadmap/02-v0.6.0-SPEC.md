### 新增组件完整规格示例：Cascader

**API 规格**:

```typescript
export interface CascaderOption {
  label: string
  value: string | number
  children?: CascaderOption[]
  disabled?: boolean
}

export interface CascaderProps {
  options: CascaderOption[]
  modelValue?: (string | number)[] // Vue
  value?: (string | number)[] // React
  placeholder?: string
  disabled?: boolean
  size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl'
  changeOnSelect?: boolean
  expandTrigger?: 'hover' | 'click'
  multiple?: boolean
  maxTagCount?: number
  checkStrictly?: boolean
  showSearch?: boolean
  filterOption?: (input: string, option: CascaderOption) => boolean
  clearable?: boolean
  virtual?: boolean
  virtualHeight?: number
}
```

**Vue 使用**:

```vue
<TiCascader
  v-model="selectedArea"
  :options="areaOptions"
  placeholder="选择地区"
  expand-trigger="hover"
  show-search
  @change="handleAreaChange" />
```

**React 使用**:

```tsx
<Cascader
  value={selectedArea}
  onChange={setSelectedArea}
  options={areaOptions}
  placeholder="选择地区"
  expandTrigger="hover"
  showSearch
/>
```

**键盘导航**: ArrowDown/Up（当前级上下移动）、ArrowRight（展开下级）、ArrowLeft（收起）、Enter（确认）、Escape（关闭）

**a11y**: `role="option"`, `aria-selected`, `aria-disabled`, `aria-live="polite"`

## B. Table 重大升级

在 v0.5.0 基础上追加：

| 升级项       | 目标               |
| ------------ | ------------------ |
| 虚拟滚动     | 10k+ 行不卡顿      |
| 行编辑       | 单行 / 批量编辑    |
| 高级筛选     | 复杂条件、范围筛选 |
| 列拖拽排序   | 鼠标拖拽调整列顺序 |
| 自定义汇总行 | 合计/平均值等汇总  |
| 行分组       | 按列值分组         |
| 导出         | CSV、基础 Excel    |

**新增 Props**:

```typescript
interface TableProps<T = any> {
  // v0.5.0 基础 props ...

  // v0.6.0 新增
  virtual?: boolean
  virtualHeight?: number
  virtualItemHeight?: number | ((index: number) => number)

  editable?: boolean
  editableCells?: Map<string, Set<number>>
  onCellChange?: (rowIndex: number, columnKey: string, newValue: any) => void

  filterMode?: 'basic' | 'advanced'
  advancedFilterRules?: FilterRule[]

  columnDraggable?: boolean
  onColumnOrderChange?: (orderedColumns: ColumnDefinition<T>[]) => void

  summaryRow?: { show: boolean; data: Record<string, any> }

  groupBy?: string
  groupRender?: (groupKey: string, items: T[]) => VNode | ReactNode

  exportable?: boolean
  exportFilename?: string
}

interface FilterRule {
  column: string
  operator: 'equals' | 'contains' | 'gt' | 'lt' | 'between'
  value: any
  logic?: 'and' | 'or'
}
```

**性能指标**: 初始渲染(10k 行虚拟) < 200ms，滚动 >= 60 FPS，内存(10k 行) < 50MB

## C. Form 重大升级

| 升级项       | 目标                             |
| ------------ | -------------------------------- |
| 动态字段     | 运行时增删字段，自动更新校验     |
| 异步校验     | 自定义异步校验函数，验证中状态   |
| 字段联动     | 字段间值依赖和条件显示           |
| 步骤表单增强 | 条件跳步、自动保存               |
| 字段级错误   | 灵活布局（inline, popup, block） |
| 重置增强     | 部分重置、撤销/重做              |

**新增 API**:

```typescript
interface FormProps {
  // v0.5.0 基础 ...

  dynamicFields?: boolean
  fieldDependencies?: Map<string, string[]>
  asyncValidators?: Record<string, (value: any) => Promise<boolean>>
}

// 表单实例方法扩展
interface FormInstance {
  // v0.5.0 基础 (validate, reset, getFieldValue, setFieldValue) ...

  addField: (field: FormField) => void
  removeField: (fieldName: string) => void
  validateFieldAsync: (fieldName: string) => Promise<{ valid: boolean; error?: string }>
  getFieldDependencies: (fieldName: string) => string[]
  undo: () => void
  redo: () => void
}
```
