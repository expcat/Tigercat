## A. 预设主题体系

### A.1 五套主题定位

| #   | 主题名           | 设计定位 | 配色特点               | 适用场景                       |
| --- | ---------------- | -------- | ---------------------- | ------------------------------ |
| 1   | **Default**      | 现代简洁 | 蓝色为主，清爽中立     | 企业应用、SaaS、后台管理       |
| 2   | **Vibrant**      | 年轻活跃 | 多彩渐变，充满能量     | 消费应用、社交、内容平台       |
| 3   | **Professional** | 商务稳重 | 深灰、深蓝为主，高级感 | 金融、医疗、企业官网           |
| 4   | **Minimal**      | 极简美学 | 黑白灰为主，高对比     | 创意工具、设计应用、开发者工具 |
| 5   | **Natural**      | 温暖亲切 | 大地色、绿色、米色系   | 教育、生活方式、内容管理       |

### A.3 主题应用方式

**HTML class**:

```html
<div class="theme-vibrant">
  <button class="ti-button">Vibrant 主题</button>
</div>
```

**JavaScript API**:

```typescript
import { ThemeManager } from '@expcat/tigercat-core'

ThemeManager.setTheme('vibrant')
ThemeManager.defineTheme('custom', {
  colors: { primary: { 500: '#ff0000' } }
})
```

**Tailwind Config**:

```javascript
import { tigercatPlugin } from '@expcat/tigercat-core/tailwind'

export default {
  plugins: [
    tigercatPlugin({
      theme: 'vibrant',
      themes: ['default', 'vibrant', 'professional', 'minimal', 'natural']
    })
  ]
}
```

## B. 组件视觉全面升级

### B.1 升级维度

| 维度       | 描述                                      | 影响组件               |
| ---------- | ----------------------------------------- | ---------------------- |
| 颜色对比度 | WCAG AA 对比度标准                        | 所有组件               |
| 间距与排版 | 优化 padding/margin/font size/line height | 所有组件               |
| 圆角统一   | 使用标准 Token 圆角值                     | 所有组件               |
| 阴影与深度 | 分层阴影营造视觉深度                      | Card, Modal, Drawer 等 |
| 微交互     | hover/focus/active 状态平滑过渡           | 所有交互组件           |
| 空状态     | 无数据、加载中等特殊状态                  | Table, List 等         |
| 禁用状态   | 统一的灰化、低对比度                      | 所有可交互组件         |

### B.2 优先升级组件列表

| 优先级 | 组件                                                                         |
| ------ | ---------------------------------------------------------------------------- |
| P0     | Button, Input, Select, Modal, Card, Table                                    |
| P1     | Form, Tabs, Menu, Pagination, Alert, Badge, Tag, Tooltip/Popover, Breadcrumb |
| P2     | Avatar, Progress, Slider, Checkbox/Radio, Switch, Tree, Dropdown, Steps      |

## D. 暗色模式全覆盖

**当前状态** (v0.4.3): 支持 `.dark` class 切换，部分组件暗色适配不完整。

**v0.7.0 目标**:

- 所有组件都有完整暗色视觉
- 自动暗色检测 (`prefers-color-scheme`)
- 完整暗色 Token 体系

**暗色 Token 示例**:

```json
{
  "light": {
    "bg-primary": "#ffffff",
    "bg-secondary": "#f9fafb",
    "text-primary": "#111827",
    "text-secondary": "#6b7280",
    "border": "#e5e7eb"
  },
  "dark": {
    "bg-primary": "#111827",
    "bg-secondary": "#1f2937",
    "text-primary": "#f9fafb",
    "text-secondary": "#d1d5db",
    "border": "#374151"
  }
}
```
