## 0. 版本概览

v0.8.0 进入**业务组件层**，专注于解决复杂、具体的业务场景。这个版本的核心是建立统一的拖拽系统、引入高级交互组件、并推出若干开箱即用的业务组件（如 RichTextEditor、Kanban、FileManager 等）。

**主要成果指标**:

- 新增 8-12 个高级/业务组件
- 建立统一的拖拽系统（列表排序、看板拖拽、面板调整大小）
- 现有交互组件（Modal, Drawer, Table）增加高级功能（拖拽移动、调整大小）
- 性能优化专项：虚拟化全覆盖、Bundle Size 优化
- 推出 CLI 脚手架工具、代码生成器

**预期版本后的组件总数**: 95-105 个（相比 v0.7.0 的 85-95，增长 10%）

## A. 拖拽系统设计与实现

### A.1 拖拽系统的目标

建立一个**统一的、可复用的、框架无关的拖拽抽象层**，支持以下应用场景：

1. **列表项排序** — 拖拽列表项重新排列
2. **看板卡片拖拽** — 跨列看板卡片移动
3. **树形节点拖拽** — Tree 组件中的拖拽排序和移动
4. **面板调整大小** — Drawer、Modal、Splitter 的尺寸调整
5. **列拖拽排序** — Table 列的位置调整

### A.2 拖拽系统的架构

```
Headless Drag Logic (框架无关)
    ↓
Vue Composable (useComposeableDrag)
React Hook (useDrag, useDrop)
    ↓
具体组件实现 (List, Kanban, Tree, Table, Modal, etc.)
```

**Headless 层实现** (`core/utils/drag.ts`):

```typescript
export interface DragItem {
  id: string | number
  data?: any
  index: number
}

export interface DragConfig {
  handleSelector?: string // 拖拽手柄选择器
  dragClass?: string // 拖拽中的 CSS class
  ghostClass?: string // 幽灵元素 class
  scrollSpeed?: number // 自动滚动速度
  scrollMargin?: number // 滚动触发的边距
}

export interface DragState {
  isDragging: boolean
  draggedItem: DragItem | null
  hoveredItem: DragItem | null
  sourceIndex: number
  targetIndex: number
}

/**
 * Headless 拖拽逻辑
 * 返回 state 和 handlers，供 Vue/React 层消费
 */
export function useHeadlessDrag(
  items: DragItem[],
  config: DragConfig = {}
): {
  state: Reactive<DragState>
  onDragStart: (item: DragItem, e: DragEvent) => void
  onDragOver: (item: DragItem, e: DragEvent) => void
  onDrop: (e: DragEvent) => void
  onDragEnd: () => void
  getReorderedItems: () => DragItem[]
} {
  // 实现拖拽逻辑，包括：
  // 1. 记录拖拽开始的项
  // 2. 计算 drag over 时的目标位置
  // 3. 处理自动滚动
  // 4. 返回重新排序后的列表
}
```

**Vue 使用示例** (Composable):

```typescript
// vue/composables/useDrag.ts

export function useDrag(items: Ref<DragItem[]>, options: DragConfig = {}) {
  const { state, onDragStart, onDragOver, onDrop, onDragEnd, getReorderedItems } = useHeadlessDrag(
    items.value,
    options
  )

  const sortedItems = computed(() => getReorderedItems())

  return {
    state,
    handlers: {
      onDragStart,
      onDragOver,
      onDrop,
      onDragEnd
    },
    sortedItems
  }
}
```

**React 使用示例** (Hook):

```typescript
// react/hooks/useDrag.ts

export function useDrag(items: DragItem[], options: DragConfig = {}) {
  const [state, setState] = useState<DragState>(...);
  const { state: headlessState, onDragStart, ... } = useHeadlessDrag(items, options);

  useEffect(() => {
    setState(headlessState);
  }, [headlessState]);

  return {
    state,
    handlers: { onDragStart, ... },
  };
}
```

### B.2 组件规格示例：RichTextEditor

**功能说明**:

RichTextEditor 是一个功能完整的富文本编辑器，基于可配置的工具栏和快捷键支持。支持多种格式化选项、文件上传、以及多种输出格式（HTML、Markdown 等）。

**关键 Props**:

```typescript
interface RichTextEditorProps {
  // 内容相关
  modelValue?: string;
  placeholder?: string;
  mode?: 'html' | 'markdown' | 'plain';

  // 功能配置
  toolbar?: ToolbarButton[];
  enableFileUpload?: boolean;
  enableLinkDialog?: boolean;
  enableTableInsert?: boolean;
  enableCodeBlock?: boolean;

  // 外观
  height?: number | string;
  readonly?: boolean;
  disabled?: boolean;

  // 事件
  @change?: (content: string) => void;
  @upload?: (file: File) => Promise<string>;  // 返回文件 URL
}

interface ToolbarButton {
  name: string;  // 'bold', 'italic', 'underline', 'link', 'image', 'codeblock', etc.
  label: string;
  icon?: string;
  tooltip?: string;
  hotkey?: string;  // 'Ctrl+B', 'Cmd+I', etc.
}
```

**使用示例**:

```vue
<template>
  <TiRichTextEditor
    v-model="content"
    :height="400"
    :toolbar="customToolbar"
    :enable-file-upload="true"
    @upload="handleFileUpload"
    @change="handleContentChange" />
</template>

<script setup lang="ts">
import { ref } from 'vue'

const content = ref('')
const customToolbar = [
  { name: 'bold', label: '粗体', hotkey: 'Ctrl+B' },
  { name: 'italic', label: '斜体', hotkey: 'Ctrl+I' },
  { name: 'link', label: '链接' },
  { name: 'codeblock', label: '代码块' },
  { name: 'image', label: '图片', hotkey: 'Ctrl+Shift+I' }
]

const handleFileUpload = async (file: File) => {
  // 上传文件，返回 URL
  const formData = new FormData()
  formData.append('file', file)
  const response = await fetch('/api/upload', { method: 'POST', body: formData })
  const data = await response.json()
  return data.url
}
</script>
```
