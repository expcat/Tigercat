## A. 拖拽系统

### A.1 设计目标

建立统一的、框架无关的 Headless 拖拽抽象层，支持：

1. 列表项排序
2. 看板卡片跨列拖拽
3. 树形节点拖拽
4. 面板调整大小（Drawer, Modal, Splitter）
5. Table 列/行拖拽排序

### A.2 架构

```
Headless Drag Logic (core/utils/drag.ts，框架无关)
    ↓
Vue Composable (useDrag)  /  React Hook (useDrag, useDrop)
    ↓
具体组件实现 (List, Kanban, Tree, Table, Modal 等)
```

### A.3 Headless 层 API

```typescript
export interface DragItem {
  id: string | number
  data?: any
  index: number
}

export interface DragConfig {
  handleSelector?: string
  dragClass?: string
  ghostClass?: string
  scrollSpeed?: number
  scrollMargin?: number
}

export interface DragState {
  isDragging: boolean
  draggedItem: DragItem | null
  hoveredItem: DragItem | null
  sourceIndex: number
  targetIndex: number
}

export function useHeadlessDrag(
  items: DragItem[],
  config?: DragConfig
): {
  state: Reactive<DragState>
  onDragStart: (item: DragItem, e: DragEvent) => void
  onDragOver: (item: DragItem, e: DragEvent) => void
  onDrop: (e: DragEvent) => void
  onDragEnd: () => void
  getReorderedItems: () => DragItem[]
}
```

### RichTextEditor 规格

```typescript
interface RichTextEditorProps {
  modelValue?: string // Vue
  value?: string // React
  placeholder?: string
  mode?: 'html' | 'markdown' | 'plain'
  toolbar?: ToolbarButton[]
  enableFileUpload?: boolean
  enableLinkDialog?: boolean
  enableTableInsert?: boolean
  enableCodeBlock?: boolean
  height?: number | string
  readonly?: boolean
  disabled?: boolean
}

interface ToolbarButton {
  name: string // 'bold', 'italic', 'link', 'image', 'codeblock' 等
  label: string
  icon?: string
  tooltip?: string
  hotkey?: string // 'Ctrl+B', 'Cmd+I' 等
}
```

**Vue 使用**:

```vue
<TiRichTextEditor
  v-model="content"
  :height="400"
  :toolbar="customToolbar"
  :enable-file-upload="true"
  @upload="handleFileUpload"
  @change="handleContentChange" />
```

## D. CLI 脚手架工具

创建 `@expcat/tigercat-cli`：

```bash
tigercat create my-app --template vue3   # 创建新项目
tigercat generate component Button       # 生成组件框架
tigercat playground                      # 启动 Playground
tigercat generate docs                   # 生成 API 文档
```

附加工具：

- **Playground 增强**: 实时编辑 Vue/React 代码、切换主题、导出到 CodePen
- **代码生成器**: Form 生成器、Table 生成器、主题生成器
