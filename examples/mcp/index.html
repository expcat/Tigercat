Tigercat MCP 技能源

本站是 @expcat/tigercat-mcp 默认读取的静态技能源：context7 组件清单 + skills markdown 文档，随每次 release 自动部署。

快速开始

Claude Code 一行接入（默认远程读取本站，无需克隆仓库）：

claude mcp add tigercat -- npx -y @expcat/tigercat-mcp

或在任意 MCP 客户端的 mcpServers 配置中加入：

{
  "mcpServers": {
    "tigercat": {
      "command": "npx",
      "args": ["-y", "@expcat/tigercat-mcp"]
    }
  }
}

本地 / 镜像模式（离线开发或 GitHub Pages 不可达时）：

tigercat-mcp --root /path/to/Tigercat        # 读取本地仓库 checkout
tigercat-mcp --base-url https://mirror.example.com/mcp/   # 指向镜像站
TIGERCAT_MCP_BASE_URL=…                      # 环境变量等价于 --base-url
tigercat-mcp --doctor                        # 校验技能源可达性与完整性

MCP 能力

  • tigercat_search
    模糊搜索组件、别名、分类、主题与命令式 API。
  • tigercat_component
    精确返回组件元数据、导入路径、props、示例与框架差异说明。
  • tigercat_route
    把自然语言任务路由到最小可用的组件 / 主题引用集合。
  • tigercat_reference
    直接读取 allow-list 内的 skill reference，支持字节截断。

另提供 tigercat://inventory 等 resources 与 tigercat-usage prompt。

技能源文件

  • context7.json 组件清单与路由映射（含 skill_files 清单）
  • skills/tigercat/SKILL.md 技能入口与路由规则
  • skills/tigercat/references/component-index.md 149+ 组件索引
  • version.json 部署版本与构建信息
