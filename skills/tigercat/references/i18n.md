# 国际化 (i18n)

通过 `locale` prop 自定义组件显示文本。

## 支持的组件

| 组件       | 主要文本             |
| ---------- | -------------------- |
| Pagination | 页码、跳转、每页条数 |
| DatePicker | 月份、星期、占位符   |
| TimePicker | 时/分/秒标签、占位符 |
| Modal      | 确定/取消按钮        |
| Drawer     | 关闭 aria-label      |
| Upload     | 拖拽提示、文件状态   |
| FormWizard | 上一步/下一步/完成   |

---

## 配置项速查

### Pagination

| 配置项            | 说明     | 中文示例        | 英文默认              |
| ----------------- | -------- | --------------- | --------------------- |
| totalText         | 总数文本 | `共 {total} 条` | `Total {total} items` |
| itemsPerPageText  | 每页后缀 | `条/页`         | `/ page`              |
| jumpToText        | 跳转标签 | `跳至`          | `Go to`               |
| pageText          | 页文本   | `页`            | `page`                |
| prevPageAriaLabel | 上一页   | `上一页`        | `Previous page`       |
| nextPageAriaLabel | 下一页   | `下一页`        | `Next page`           |

### DatePicker

| 配置项         | 说明                   | 中文示例   |
| -------------- | ---------------------- | ---------- |
| locale         | Intl locale 代码       | `zh-CN`    |
| today          | 今天按钮               | `今天`     |
| ok             | 确定按钮               | `确定`     |
| calendar       | 日历 dialog aria 文案  | `日历`     |
| toggleCalendar | 打开日历按钮 aria 文案 | `打开日历` |
| clearDate      | 清除按钮 aria 文案     | `清除日期` |
| previousMonth  | 上个月按钮 aria 文案   | `上个月`   |
| nextMonth      | 下个月按钮 aria 文案   | `下个月`   |

### TimePicker

| 配置项                                | 说明       | 中文示例           |
| ------------------------------------- | ---------- | ------------------ |
| hourLabel / minuteLabel / secondLabel | 时分秒标签 | `时` / `分` / `秒` |
| placeholder                           | 占位符     | `选择时间`         |
| now                                   | 此刻按钮   | `此刻`             |
| confirm                               | 确定按钮   | `确定`             |

### Modal / Drawer

| 配置项         | 说明     | 中文示例 |
| -------------- | -------- | -------- |
| okText         | 确定按钮 | `确定`   |
| cancelText     | 取消按钮 | `取消`   |
| closeAriaLabel | 关闭标签 | `关闭`   |

### Upload

| 配置项        | 说明     | 中文示例         |
| ------------- | -------- | ---------------- |
| dragText      | 拖拽提示 | `将文件拖到此处` |
| clickText     | 点击文本 | `点击上传`       |
| uploadingText | 上传中   | `上传中...`      |
| successText   | 成功     | `上传成功`       |
| errorText     | 失败     | `上传失败`       |
| retryText     | 重试     | `重试`           |

### FormWizard

| 配置项     | 说明           | 中文示例 |
| ---------- | -------------- | -------- |
| prevText   | 上一步按钮文案 | `上一步` |
| nextText   | 下一步按钮文案 | `下一步` |
| finishText | 完成按钮文案   | `完成`   |

## 预设值导入

```ts
import { ZH_CN_PAGINATION_LABELS } from '@expcat/tigercat-core'
import { ZH_CN_DATEPICKER_LOCALE } from '@expcat/tigercat-core/datepicker-locales/zh-CN'
```

---

## 使用方式

### 组件级配置

```vue
<!-- Vue -->
<Pagination :total="100" :locale="{ pagination: ZH_CN_PAGINATION_LABELS }" />
<DatePicker :locale="{ datePicker: ZH_CN_DATEPICKER_LOCALE }" />
```

```tsx
// React
<Pagination total={100} locale={{ pagination: ZH_CN_PAGINATION_LABELS }} />
<DatePicker locale={{ datePicker: ZH_CN_DATEPICKER_LOCALE }} />
```

### 全局配置

```ts
// Vue: app.provide('tigercat-locale', locale)
// React: <TigercatProvider locale={locale}>

const locale = {
  pagination: ZH_CN_PAGINATION_LABELS,
  datePicker: ZH_CN_DATEPICKER_LOCALE,
  formWizard: {
    prevText: '上一步',
    nextText: '下一步',
    finishText: '完成'
  }
}
```

## 内置语言包（按需加载）

Tigercat 内置 8 种语言包：`en-US`、`zh-CN`、`zh-TW`、`ja-JP`、`ko-KR`、`th-TH`、`vi-VN`、`id-ID`。

DatePicker 专用文案也提供同名子路径，只导入对应语言的日期面板文案：

```ts
import { ZH_CN_DATEPICKER_LOCALE } from '@expcat/tigercat-core/datepicker-locales/zh-CN'
```

**推荐**通过子路径按需引入，避免把未使用的语言打进生产包：

```ts
// 仅引入中文，未使用的语言会被 bundler tree-shake 掉
import { zhCN } from '@expcat/tigercat-core/locales/zh-CN'
import { jaJP } from '@expcat/tigercat-core/locales/ja-JP'
```

主入口仍兼容传统写法（向后兼容，但会引入全部内置语言）：

```ts
import { zhCN, enUS } from '@expcat/tigercat-core'
```

### Vue 全局配置

```vue
<script setup lang="ts">
import { ConfigProvider } from '@expcat/tigercat-vue'
import { zhCN } from '@expcat/tigercat-core/locales/zh-CN'
</script>

<template>
  <ConfigProvider :locale="zhCN">
    <App />
  </ConfigProvider>
</template>
```

### React 全局配置

```tsx
import { ConfigProvider } from '@expcat/tigercat-react'
import { zhCN } from '@expcat/tigercat-core/locales/zh-CN'

export function Root() {
  return (
    <ConfigProvider locale={zhCN}>
      <App />
    </ConfigProvider>
  )
}
```

### 异步懒加载（适合多语言切换）

```ts
async function loadLocale(lang: 'zh-CN' | 'ja-JP' | 'ko-KR') {
  switch (lang) {
    case 'zh-CN':
      return (await import('@expcat/tigercat-core/locales/zh-CN')).zhCN
    case 'ja-JP':
      return (await import('@expcat/tigercat-core/locales/ja-JP')).jaJP
    case 'ko-KR':
      return (await import('@expcat/tigercat-core/locales/ko-KR')).koKR
  }
}
```

## 自定义语言包

```ts
import type { TigerLocale } from '@expcat/tigercat-core'

const myLocale: Partial<TigerLocale> = {
  pagination: {
    totalText: '全部 {total} 筆',
    jumpToText: '前往'
  },
  datePicker: {
    placeholder: '選擇日期',
    today: '今天'
  },
  modal: {
    okText: '確定',
    cancelText: '取消'
  }
}
```
