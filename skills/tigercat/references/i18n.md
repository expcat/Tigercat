# 国际化 (i18n)

Tigercat 组件支持国际化配置，通过 `locale` prop 自定义显示文本。

## 支持的组件

| 组件       | 支持程度 | 主要文本                       |
| ---------- | -------- | ------------------------------ |
| Pagination | ✅ 完整  | 页码、跳转、每页条数           |
| DatePicker | ✅ 完整  | 月份、星期、占位符             |
| TimePicker | ✅ 完整  | 时/分/秒标签、占位符           |
| Modal      | ✅ 部分  | 确定/取消按钮、关闭 aria-label |
| Drawer     | ✅ 部分  | 关闭按钮 aria-label            |
| Upload     | ✅ 完整  | 拖拽提示、文件状态、按钮       |

## 类型定义

```ts
import type { TigerLocale } from '@expcat/tigercat-core'

// locale prop 类型
type LocaleProp = Partial<TigerLocale>
```

## Pagination

### 配置项

```ts
interface TigerLocalePagination {
  totalText?: string // 总数文本，支持 {total}, {start}, {end}
  itemsPerPageText?: string // 每页条数后缀
  jumpToText?: string // 跳转标签
  pageText?: string // "页" 文本
  prevPageAriaLabel?: string // 上一页 aria-label
  nextPageAriaLabel?: string // 下一页 aria-label
  pageAriaLabel?: string // 页码 aria-label，支持 {page}
}
```

### 预设值

```ts
import { ZH_CN_PAGINATION_LABELS } from '@expcat/tigercat-core'

// 中文简体
const ZH_CN = {
  totalText: '共 {total} 条',
  itemsPerPageText: '条/页',
  jumpToText: '跳至',
  pageText: '页',
  prevPageAriaLabel: '上一页',
  nextPageAriaLabel: '下一页',
  pageAriaLabel: '第 {page} 页'
}

// 英文（默认）
const EN_US = {
  totalText: 'Total {total} items',
  itemsPerPageText: '/ page',
  jumpToText: 'Go to',
  pageText: 'page',
  prevPageAriaLabel: 'Previous page',
  nextPageAriaLabel: 'Next page',
  pageAriaLabel: 'Page {page}'
}
```

### 使用

```vue
<!-- Vue -->
<Pagination :total="100" :locale="{ pagination: ZH_CN_PAGINATION_LABELS }" />
```

```tsx
// React
<Pagination total={100} locale={{ pagination: ZH_CN_PAGINATION_LABELS }} />
```

## DatePicker

### 配置项

```ts
interface DatePickerLocale {
  months?: string[] // 月份名称 (1-12)
  monthsShort?: string[] // 月份简写
  weekdays?: string[] // 星期名称 (周日-周六)
  weekdaysShort?: string[] // 星期简写
  weekdaysMin?: string[] // 星期最短形式

  endPlaceholder?: string // 范围选择结束占位符
```

## 全局配置

### Vue 3

```ts
// main.ts
import { createApp } from 'vue'
import { TigercatProvider } from '@expcat/tigercat-vue'
import { ZH_CN_PAGINATION_LABELS, ZH_CN_DATEPICKER_LOCALE } from '@expcat/tigercat-core'

const locale = {
  pagination: ZH_CN_PAGINATION_LABELS,
  datePicker: ZH_CN_DATEPICKER_LOCALE
}

const app = createApp(App)
app.provide('tigercat-locale', locale)
app.mount('#app')
```

### React

```tsx
// App.tsx
import { TigercatProvider } from '@expcat/tigercat-react'
import { ZH_CN_PAGINATION_LABELS, ZH_CN_DATEPICKER_LOCALE } from '@expcat/tigercat-core'

const locale = {
  pagination: ZH_CN_PAGINATION_LABELS,
  datePicker: ZH_CN_DATEPICKER_LOCALE
}

function App() {
  return (
    <TigercatProvider locale={locale}>
      <MyApp />
    </TigercatProvider>
  )
}
```
