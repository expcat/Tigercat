# Tigercat React

React examples use controlled props, camelCase callbacks, and JSX syntax. Install from `@expcat/tigercat-react`; import components from PascalCase subpaths for on-demand usage.

查组件用法：先开 [component-index.md](../component-index.md) 定位组件、Category 与 PascalCase Package Subpath，再按其规则打开 `shared/props/{cat}.md` 与 `examples/{cat}.md`。跨框架绑定差异见 [shared/patterns/common.md](../shared/patterns/common.md) 与 [shared/glossary.md](../shared/glossary.md)。
